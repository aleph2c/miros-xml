Miros-XML#

The miros-xml is a Python library which: consumes XML files describing a statechart and turns it into a working python program:

  • The parallel regions feature will be supported.

  • Its XML mapping is inspired by the SCXML standard.

The majority of this document is written to describe how to implement parallel regions using the miros event processor.
